# NovaChat

NovaChat is a real-time chat application built using the MERN stack and Socket.IO.

## Tech Stack
- MongoDB
- Express.js
- React
- Node.js
- Socket.IO

## Features
- Real-time messaging
- User connection events
- Broadcasting messages
- Scalable WebSocket communication

## Purpose
This project was built to practice real-time application architecture and WebSocket communication.
